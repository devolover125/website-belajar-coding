Apa itu CSS?

CSS (Cascading Style Sheets) adalah bahasa yang digunakan untuk mendesain tampilan halaman web. CSS mengontrol tata letak, warna, font, dan berbagai aspek lain dari tampilan halaman web.

Struktur Dasar CSS

Berikut adalah contoh dasar aturan CSS:

h1 {
    color: blue;
    font-size: 24px;
    text-align: center;
}

Elemen dan Properti CSS

1. Selektor CSS

Selektor CSS digunakan untuk memilih elemen HTML tertentu untuk diberikan gaya. Ada beberapa jenis selektor, termasuk:

  • Selektor Elemen: Memilih semua elemen tertentu. Misalnya, memilih semua elemen <p>.
  • Selektor ID: Memilih elemen berdasarkan ID yang bersifat unik.
  • Selektor Kelas: Memilih elemen berdasarkan kelas yang dapat diterapkan ke beberapa elemen.
/* Selektor Elemen */
p {
    color: green;
}

/* Selektor ID */
#main-title {
    color: red;
}

/* Selektor Kelas */
.box {
    border: 1px solid black;
}

2. Properti CSS Dasar

Properti CSS digunakan untuk menentukan gaya. Berikut adalah beberapa properti dasar yang sering digunakan:

  • color: Mengatur warna teks.
  • font-size: Mengatur ukuran teks.
  • margin: Mengatur jarak luar elemen.
  • padding: Mengatur jarak dalam elemen.
  • background-color: Mengatur warna latar belakang.
div {
    background-color: lightgrey;
    padding: 10px;
    margin: 20px;
    border: 2px solid black;
}

3. Responsivitas dengan Media Queries

Media queries digunakan untuk membuat website responsif terhadap berbagai ukuran layar. Contoh:

@media (max-width: 600px) {
    body {
        background-color: lightblue;
    }
}

4. Flexbox

Flexbox adalah teknik layout yang membuat tata letak elemen lebih fleksibel. Contoh penggunaan Flexbox:

.container {
    display: flex;
    justify-content: space-between;
    align-items: center;
}

5. Grid Layout

Grid Layout adalah sistem tata letak yang berbasis grid dengan baris dan kolom. Contoh Grid Layout:

.container {
    display: grid;
    grid-template-columns: repeat(3, 1fr);
    grid-gap: 10px;
}

6. Transisi dan Animasi

Transisi dan animasi digunakan untuk membuat perubahan gaya menjadi lebih halus atau menciptakan animasi sederhana. Contoh:

button {
    transition: background-color 0.5s;
}

button:hover {
    background-color: red;
}

/* Animasi */
@keyframes fadeIn {
    from { opacity: 0; }
    to { opacity: 1; }
}

div {
    animation: fadeIn 2s;
}

Kesimpulan

CSS adalah alat yang kuat untuk mengontrol tampilan halaman web. Dengan menggunakan selektor, properti, dan teknik layout seperti Flexbox dan Grid, kita bisa membuat desain web yang responsif dan menarik.
