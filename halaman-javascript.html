Apa itu JavaScript?

JavaScript adalah bahasa pemrograman yang digunakan untuk membuat halaman web interaktif. Dengan JavaScript, kamu dapat memanipulasi elemen HTML, menangani peristiwa, dan berkomunikasi dengan server.

Menambahkan JavaScript ke Halaman Web

JavaScript dapat ditambahkan ke halaman HTML dengan menggunakan tag <script>. Berikut adalah contoh cara menambahkannya di dalam tag <head> atau sebelum penutup tag </body>:

<script src="script.js"></script>

Dasar-Dasar JavaScript

1. Variabel

Variabel digunakan untuk menyimpan data. Dalam JavaScript, kamu bisa mendeklarasikan variabel menggunakan var, let, atau const.

let nama = "Aldi";
const umur = 25;

2. Tipe Data

JavaScript memiliki beberapa tipe data, antara lain:

  • String: Teks, contoh: "Hello"
  • Number: Angka, contoh: 123
  • Boolean: Nilai benar atau salah, contoh: true, false
  • Array: Kumpulan nilai, contoh: [1, 2, 3]
  • Object: Kumpulan pasangan kunci-nilai, contoh: { nama: "Aldi", umur: 25 }

3. Struktur Kontrol

JavaScript mendukung struktur kontrol seperti if, for, dan while. Berikut adalah contoh penggunaan if:

if (umur >= 18) {
    console.log("Dewasa");
} else {
    console.log("Belum Dewasa");
}

4. Fungsi

Fungsi adalah blok kode yang dapat dipanggil untuk menjalankan tugas tertentu. Berikut adalah contoh fungsi sederhana:

function sapa(nama) {
    return "Halo, " + nama + "!";
}

console.log(sapa("Aldi"));

Manipulasi DOM

JavaScript memungkinkan kamu untuk memanipulasi elemen HTML di halaman. Contoh mengubah teks dalam elemen dengan ID tertentu:

document.getElementById("judul").innerText = "Judul Baru";

Event Handling

JavaScript dapat menangani peristiwa, seperti klik pada tombol. Berikut adalah contoh penggunaan event:

document.getElementById("tombol").addEventListener("click", function() {
    alert("Tombol diklik!");
});

Kesimpulan

JavaScript adalah bahasa yang sangat kuat untuk membuat halaman web interaktif. Dengan memahami dasar-dasar JavaScript, kamu dapat mulai membuat aplikasi web yang menarik dan dinamis. Selanjutnya, kamu dapat menerapkan pengetahuan ini dalam proyek web yang lebih besar.
