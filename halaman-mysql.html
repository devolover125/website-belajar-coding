Apa itu MySQL?

MySQL adalah sistem manajemen basis data relasional (RDBMS) yang menggunakan Structured Query Language (SQL). MySQL sering digunakan untuk menyimpan data aplikasi web.

Instalasi MySQL

MySQL dapat diinstal di berbagai sistem operasi. Untuk menginstal MySQL, kamu bisa mengunduh dari situs resmi MySQL dan mengikuti petunjuk instalasi.

Membuat Database dan Tabel

Untuk membuat database baru, kamu dapat menggunakan perintah CREATE DATABASE:

CREATE DATABASE nama_database;

Setelah membuat database, kamu bisa membuat tabel dengan perintah CREATE TABLE:

CREATE TABLE nama_tabel (
    id INT AUTO_INCREMENT PRIMARY KEY,
    nama VARCHAR(100),
    umur INT
);

Operasi Dasar CRUD

1. Create (Menambah Data)

Untuk menambah data ke tabel, gunakan perintah INSERT INTO:

INSERT INTO nama_tabel (nama, umur) VALUES ('Aldi', 25);

2. Read (Membaca Data)

Untuk membaca data dari tabel, gunakan perintah SELECT:

SELECT * FROM nama_tabel;

3. Update (Memperbarui Data)

Untuk memperbarui data, gunakan perintah UPDATE:

UPDATE nama_tabel SET umur = 26 WHERE nama = 'Aldi';

4. Delete (Menghapus Data)

Untuk menghapus data, gunakan perintah DELETE:

DELETE FROM nama_tabel WHERE nama = 'Aldi';

Query dan Penggunaan JOIN

Untuk menggabungkan data dari dua tabel, kamu bisa menggunakan JOIN. Berikut adalah contoh penggunaan INNER JOIN:

SELECT a.nama, b.nama_kota
FROM pengguna a
INNER JOIN kota b ON a.kota_id = b.id;

Penggunaan Kondisi dan Pengurutan

Untuk menambahkan kondisi dalam query, gunakan WHERE. Untuk mengurutkan hasil, gunakan ORDER BY:

SELECT * FROM nama_tabel
WHERE umur > 20
ORDER BY nama ASC;

Kesimpulan

MySQL adalah sistem manajemen basis data yang kuat dan banyak digunakan dalam pengembangan aplikasi web. Dengan memahami dasar-dasar SQL dan operasi CRUD, kamu dapat mulai bekerja dengan database untuk menyimpan dan mengelola data aplikasi.
