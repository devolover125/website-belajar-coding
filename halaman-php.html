Apa itu PHP?

PHP (Hypertext Preprocessor) adalah bahasa pemrograman server-side yang digunakan untuk pengembangan web. PHP dapat digunakan untuk membuat aplikasi web dinamis dan berinteraksi dengan database.

Menambahkan PHP ke Halaman Web

File PHP biasanya memiliki ekstensi .php. Kode PHP ditulis di dalam tag <?php ?>. Berikut adalah contoh sederhana:

<?php
    echo "Hello, World!";
?>

Dasar-Dasar PHP

1. Variabel

Variabel dalam PHP dimulai dengan tanda dollar ($). Contoh mendeklarasikan variabel:

$nama = "Aldi";
$umur = 25;

2. Tipe Data

PHP memiliki beberapa tipe data, antara lain:

  • String: Teks, contoh: "Hello"
  • Integer: Bilangan bulat, contoh: 123
  • Float: Bilangan desimal, contoh: 12.34
  • Boolean: Nilai benar atau salah, contoh: true, false
  • Array: Kumpulan nilai, contoh: array(1, 2, 3)
  • Object: Objek dari kelas.

3. Struktur Kontrol

PHP mendukung struktur kontrol seperti if, for, dan while. Berikut adalah contoh penggunaan if:

if ($umur >= 18) {
    echo "Dewasa";
} else {
    echo "Belum Dewasa";
}

4. Fungsi

Fungsi dalam PHP dideklarasikan dengan kata kunci function. Berikut adalah contoh fungsi sederhana:

function sapa($nama) {
    return "Halo, " . $nama . "!";
}

echo sapa("Aldi");

Pengolahan Formulir

PHP sering digunakan untuk memproses data dari formulir HTML. Berikut adalah contoh sederhana:

if ($_SERVER["REQUEST_METHOD"] == "POST") {
    $nama = $_POST["nama"];
    echo "Nama: " . $nama;
}

Interaksi dengan Database

PHP dapat berinteraksi dengan database menggunakan MySQL. Berikut adalah contoh koneksi dan query sederhana:

$servername = "localhost";
$username = "username";
$password = "password";
$dbname = "database";

$conn = new mysqli($servername, $username, $password, $dbname);

// Cek koneksi
if ($conn->connect_error) {
    die("Koneksi gagal: " . $conn->connect_error);
}

$sql = "SELECT id, nama FROM users";
$result = $conn->query($sql);

if ($result->num_rows > 0) {
    while($row = $result->fetch_assoc()) {
        echo "id: " . $row["id"]. " - Nama: " . $row["nama"]. "
";
    }
} else {
    echo "0 hasil";
}

$conn->close();

Kesimpulan

PHP adalah bahasa yang kuat untuk pengembangan web dinamis. Dengan memahami dasar-dasar PHP, kamu dapat mulai membangun aplikasi web yang berinteraksi dengan pengguna dan database. Selanjutnya, kamu bisa menerapkan pengetahuan ini dalam proyek yang lebih kompleks.
